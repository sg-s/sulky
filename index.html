Sulky

TL;DR

Sulky is a application meant to run in your browser that can organize and display information in novel ways. Sulky can be used to generate a rich, inter-linked web of information about a specific topic. Sulky can generate rich documents that allow you to non-linearly read its contents. Sulky organizes information hierarchically, so you can quickly understand what a particular node of information is about, and then dig in if you want to know more.

Principles

Graph Structure

Information lives on a graph, which is a fancy way of saying a network. A graph is made up of nodes, which contain various flavors of information, and edges, which connect two nodes, and may themselves contain information.
